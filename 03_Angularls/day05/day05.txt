一、路由
	案例：
	设置3个链接，锚点值分别为#/a,#/b,#/c
	当请求地址是.....html#/ ===显示 这是首页
	当请求地址是.....html#/a ===显示 这是信息页
	当请求地址是.....html#/b ===显示 外部页面
		外部页面遍历一个数组["雪碧","可乐","橙汁"]
	当请求地址是.....html#/c ===跳转到这是首页

	将控制器中绑定的数据,修改为从后台php获取

	路由的参数
		02_params.html
		设置参数:
			在配置路由的时候,when里的请求路径,会在:后面添加参数
			配置/a  ===  只能处理 #/a的请求
			配置/a/b ===只能处理#/a/b的请求
			配置/:a  ===  能处理所有的一个层级的请求
					能处理:	/a,/b,/e,/2
					不能处理/a/b,/2/a
			配置/a/:a ===能处理所有以/a开头的2个层级的请求
					/a/a,/a/123,/a/abc
					/a,/a/b/c,/b/a
			配置/a/:b/c 
		获取参数:
			依赖$routeParams对象/模块
			使用该模块去点参数名即可获取传递的参数

三、Shell指令
	Linux系统中,目录指的就是Windows系统中的文件夹
	1. pwd
		print working directory
		打印工作的目录(显示当前所在路径)
	2. cd
		change directory
		切换工作目录
		cd a  切换到当前目录下的a目录中去
			如果当前目录没有该文件夹,则报错
		cd ..  切换到当前目录的上一级目录
			当回退到根目录(/)时,再使用该命令,无法再继续切换了
		cd d:  切换到D盘
		cd ~   切换到当前登录用户的主页面
	3. ls
		list
		显示当前目录下所有的文件和目录(非隐藏)
		linux系统隐藏文件,使用的是文件名前加.的方式
		list 显示非隐藏的文件或目录
		list -ah 显示所有的,包括隐藏的
		list -lt 显示所有非文件或目录,以列表的形式,按照时间顺序排列出来
	练习:
		在当前文件的目录下右击git bash
		查看当前路径,切换路径,查看当前路径中的文件

	4. 新建目录
		mkdir(make directory)
		mkdir 目录名  在当前目录下新建指定名称的目录
			如果目录已经存在,则创建失败
		mkdir ./a  在当前路径下新建文件夹,名字为a
		mkdir ./a/b  在当前路径下的a目录下,新建文件夹b
			如果a目录不存在,则报错
		mkdir -p a/b  在当前路径下的a目录下,创建文件夹b,
			如果a目录不存在,则先创建a目录,再创建b目录
	5. 新建文件
		touch
		touch 1.txt  在当前路径下创建1.txt
			如果文件已经存在,再次执行该命令,会修改该文件的最后访问时间
		如果touch的是一个目录,不会报错,只会修改该目录的最后访问时间

	6. 删除目录
		rmdir
		remove directory
		rmdir 目录名  删除当前目录下指定名称的目录
			只能删除空目录			
	7. 删除文件
		rm
		remove
		rm 文件名  删除当前目录下指定文件名的文件

		删除非空目录
		rm -rf 目录名

	8. vi编辑器
		文本编辑器
		vi 文件名  打开某个文件,如果文件不存在,则先创建文件,再打开该文件
		打开文件后,vi先进入的是命令模式,此时不可输入数据,但是可以切换到其他两种模式:输入模式,底行模式
		输入模式:命令模式下按a/i进入,可以输入数据,输入结束后按ESC推出输入模式,重新进入命令模式.
		底行模式:命令模式下按shift+:进入,
			w  : write,写入数据,保存数据
			q  : quit,退出vi界面
			q! : 强制退出,不保存
			wq :  保存并退出

	9. 查看文本内容
		cat
		catch
		cat 文件名 读取某个文件的内容
